What it does and how I made it happen.

"Fabriko" - An E-commerce project where I served as a Full Stack Developer. Using React for the front-end and Java Spring Suite for the back-end, supported by MySQL for database management. For styling, I harnessed the power of Bootstrap and CSS. For deployment, I used AWS EC2 and used S3 bucket to store/view images.

The platform includes distinct user roles – shoppers and administrators. Administrators have CRUD capabilities for product management, while shoppers experience seamless shopping. User authentication ensures secure access for category exploration, product viewing, and streamlined billing and shipping. Not logged in, users can still view categories/products, but cart additions and admin CRUD are restricted for security. Impressively, user carts and billing/shipping info persist even through logouts.

"Fabriko" showcases my Full Stack Developer proficiency through user roles, dynamic shopping, and data management. This project highlights my collaborative and adaptable skills, proving my ability to excel in teams and deliver impactful E-commerce solutions.
